## Features

- **Project Management**: View, create, and manage construction projects.
- **Materials Management**: Track and organize construction materials.
- **Employee Management**: Maintain a database of employees and their details.

## Technologies Used

- **Frontend**: HTML, CSS
- **Backend**: PHP
- **Database**: MySQL
